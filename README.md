# Portfolio Risk Analysis using Monte Carlo Simulation

## Objective
To estimate Value at Risk (VaR) of a diversified portfolio using Monte Carlo simulation.

## Methodology
- Historical return analysis
- Covariance & correlation estimation
- Monte Carlo simulation of correlated returns
- 95% and 99% VaR computation

## Tools Used
- Python (NumPy, Pandas)
- Portfolio Theory
- Risk Management Concepts

## Output
- One-page PDF report summarizing methodology and results
